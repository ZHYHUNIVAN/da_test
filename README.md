# Image uploading app
# API
Before starting API server, from repo root:
```bash
cd api
bundle install
rails db:create db:migrate db:seed
```
then to start the server:
```bash
rails s
```
# Frontend
Before starting frontend server, from repo root:
```bash
cd frontend
npm install
```
then to start server:
```bash
npm run dev
```
## Go to
http://localhost:8080/
